# Advanced Config Examples

These configurations demonstrate advanced features and specialized use cases that go beyond basic extraction and loading.

## Files

- `enhanced_example.yaml` - Showcases enhanced features like parallel processing, custom partitioning, and advanced error handling
- `silver_join_example.yaml` - Demonstrates joining multiple Silver datasets with configurable join strategies

## Purpose

These configs are for:
- Advanced production scenarios
- Performance optimization examples
- Complex data integration patterns
- Specialized workflow requirements

## Usage

These examples require understanding of the basic concepts and are intended for users implementing sophisticated data pipelines. They showcase the full capabilities of the framework.
