# Configuration Examples

This directory contains example configurations for medallion-foundry, organized by purpose and complexity level.

## Directory Structure

- **`examples/`** - Main example configurations referenced in documentation
- **`simple/`** - Minimal starter configs for learning basic structure
- **`complex/`** - Advanced production-ready configurations with all features
- **`patterns/`** - Examples of different data loading patterns (full, CDC, SCD)
- **`advanced/`** - Specialized configs for advanced features and complex scenarios
- **`templates/`** - Template files for creating new configurations
- **`quickstart/`** - Minimal configs for getting started quickly

## Choosing a Config

- **New to medallion-foundry?** Start with `quickstart/quick_test.yaml`
- **Learning the basics?** Use configs in `simple/`
- **Implementing production pipelines?** Reference configs in `complex/` or `examples/`
- **Need specific patterns?** Check `patterns/` for your data loading strategy
- **Advanced features?** Look in `advanced/`
- **Creating new configs?** Use templates in `templates/`

Each subdirectory contains a README explaining the specific purpose of the configs within it.
