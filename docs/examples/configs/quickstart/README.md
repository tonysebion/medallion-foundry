# Quickstart Config

This directory contains configurations for getting started quickly with medallion-foundry.

## Files

- `quick_test.yaml` - Minimal config for testing the framework setup

## Purpose

This config is designed for:
- Verifying that the framework is installed correctly
- Testing basic functionality without complex setup
- Following along with the quickstart tutorial
- Initial validation of the environment

## Usage

This is the config used in the beginner quickstart guide. It works with minimal dependencies and can be run immediately after installation to confirm everything is working.
